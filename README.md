### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/riyagupta89/hulu
    ```
2. Install NPM packages
    ```sh
    yarn install
    ```
3. Start the application
    ```sh
    yarn dev
    ```
The above command will start the application on [http://localhost:3000/](http://localhost:3000).
